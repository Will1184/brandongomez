Guía de Configuración de Angular

Prerequisitos

Node.js

v^18.9.1 o más reciente

Editor de texto

Recomendamos Visual Studio Code

Terminal

Necesaria para ejecutar los comandos de Angular CLI

Instrucciones

La siguiente guía te guiará a través de la configuración de un proyecto Angular local.

Instalar Angular CLI

Abre una terminal (si usas Visual Studio Code, puedes abrir una terminal integrada) y ejecuta el siguiente comando:

npm install -g @angular/cli

Si tienes problemas para ejecutar este comando en Windows o Unix, consulta la documentación de CLI para más información.

Crear un nuevo proyecto

En tu terminal, ejecuta el comando CLI ng new con el nombre deseado para el proyecto. En los siguientes ejemplos, usaremos el nombre de proyecto my-first-angular-app.

ng new <project-name>

Se te presentarán algunas opciones de configuración para tu proyecto. Usa las teclas de flecha y enter para navegar y seleccionar las opciones que desees.

Si no tienes preferencias, solo presiona la tecla enter para aceptar las opciones predeterminadas y continuar con la configuración.

Después de seleccionar las opciones de configuración y que el CLI complete la configuración, deberías ver el siguiente mensaje:

✔ Packages installed successfully.
Successfully initialized git.

Ejecutar tu nuevo proyecto localmente

En tu terminal, cambia a tu nuevo proyecto Angular:

cd my-first-angular-app

Todos tus dependencias deberían estar instaladas en este punto (puedes verificarlo comprobando la existencia de una carpeta node_modules en tu proyecto), por lo que puedes iniciar tu proyecto ejecutando el comando:

npm start

Si todo es exitoso, deberías ver un mensaje de confirmación similar en tu terminal:

Watch mode enabled. Watching for file changes...
                NOTE: Raw file sizes do not reflect development server per-request transformations.
                ➜ Local: http://localhost:4200/
                ➜ press h + enter to show help

Ahora puedes visitar la ruta en Local (por ejemplo, http://localhost:4200) para ver tu aplicación.
